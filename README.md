# Unified Video Shorts Pipeline

Автоматизированный пайплайн для создания, обработки и публикации коротких видео (Shorts/Reels) на YouTube, TikTok и Instagram.

## Возможности

- 🔍 **Поиск трендовых видео** по ключевым словам с фильтрацией по длительности и просмотрам
- ⬇️ **Параллельное скачивание** видео с проверкой целостности
- ✂️ **Умная нарезка** на клипы (по сценам и тишине)
- 🤖 **AI-анализ** (YOLO + Ollama) – генерация заголовков, описаний, тегов
- 🎨 **Постобработка** – наложение на фон в виде круга, клонирование с вариациями
- 📦 **Распределение** готовых видео по аккаунтам с учётом дневных лимитов
- 🚀 **Автоматическая загрузка** с симуляцией человеческого поведения (лайки, скроллинг, поиск)
- 📊 **Отчётность** в Telegram, архивирование исходников

## Установка

### Требования
- Python 3.9+
- [FFmpeg](https://ffmpeg.org/) (должен быть в PATH)
- [Ollama](https://ollama.com/) с моделью `llava` (для AI-анализа)
- Браузер Chromium (для Playwright)
